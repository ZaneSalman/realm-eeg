## [0.1.0] - 2026-07-16

Initial public source release. Scientific use and any subsequent release artifacts remain
subject to the safeguards in `RELEASE_CHECKLIST.md`.

Archived release: [10.5281/zenodo.21386115](https://doi.org/10.5281/zenodo.21386115).

### Added

- Initial unpublished research reference implementation of the REALM EEG
  framework.
- Synthetic-data examples and reproducibility-oriented project structure.
- Governance, citation, security, contribution, and release-checklist files.
- Apache-2.0 licensing for software and CC BY 4.0 licensing for original
  documentation, figures, and tables.

### Known limitations

- No completed patient-level empirical validation is claimed.
- The software is not intended or approved for clinical use.
